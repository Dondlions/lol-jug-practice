# 🏆 LOL Jungle Timer Pro

专为英雄联盟打野玩家设计的终极练习工具，采用赛事 HUD 风格悬浮窗，支持 Riot 本地 API 优先、视觉识别兜底的全自动计时！

![Version](https://img.shields.io/badge/version-2.0-blue)
![Python](https://img.shields.io/badge/python-3.7+-green)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey)

## ✨ 特性

- 🎨 **现代化UI** - 游戏风格设计，深色主题，发光按钮
- 🪟 **HUD 悬浮窗** - 半透明、置顶、默认只显示核心信息
- 🤖 **自动监控** - 优先 Riot 本地 API，失败自动回退视觉识别
- ⚡ **实时计时** - 精确到百分之一秒
- 📊 **数据统计** - 历史记录、最佳成绩、平均用时
- 🎯 **路线预设** - 红开/蓝开/全清等多种路线
- ⌨️ **快捷键** - F1开始 F2暂停 F3重置 F4完成

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install mss opencv-python numpy

# 2. 运行
cd lol_jungle_timer
python main.py
```

## 🎮 使用

### 手动模式
1. 选择打野路线
2. 点击 **▶ 开始** 或按 **F1**
3. 每清理一个营地，点击对应卡片
4. 自动或手动(F4)完成

### 全自动模式（推荐）
1. 将 HUD 放在游戏边缘
2. 勾选 **🤖 自动监控**
3. 如需视觉识别兜底，选择游戏分辨率
4. 进入LOL练习模式
5. 程序会先尝试 **Riot 本地 API**
6. 如果本地 API 不可用，会自动切到 **视觉识别**
7. **0:55自动开始 → 4级自动结束**

## 🎯 技巧

- **自动监控**: 默认先走 Riot 本地 API，更稳更准
- **视觉识别**: 当 Riot 本地 API 不可用时自动接管
- **测试识别**: 点击"▶️ 测试识别"预览效果
- **校准**: 如识别不准，点击"👁️ 校准视觉"
- **快捷键**: F4可随时随地手动完成
